Name: Slime
Summary: Item that is ran into by the player and decreases health.
Description: This prefab is for a 2D sprite.

Components:
Sprite Renderer: The Slime Gel image is used as the sprite.

Circle Collider 2D: This component is the key's collider to handle collisions with the player. This collider is a trigger.

Powerup Pickup (Script): This script accesses the player's health variable and lowers it.
