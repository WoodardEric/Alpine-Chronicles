Name: Computer
Summary: Intractable that is interacted with by the player in Level 0.
Description: This prefab is both Intractable and NPC, using NPC code for dialogue.

Components:
Sprite Renderer: The laptop-with-pixel-art-style image is used as the sprite.

Box Collider 2D: This component is the computer's collider to handle collisions with the player. This collider is not a trigger.

NPC (Script): This script manages the NPC dialogue when interacted with by the player, and utilized the NPC UI Dialogue GameObject.
