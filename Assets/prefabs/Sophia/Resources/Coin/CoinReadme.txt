Name: Coin
Summary: Collectable item used to track score
Description: This prefab is for a 2D sprite.

Components:
Sprite Renderer: The Golden Coin image is used as the sprite.

Capsule Collider 2D: This component is the coin's collider to handle collisions with the player. This collider is a trigger.

Coin Pickup (Script): This script tracks the score, and has an attached audio source for when the item is picked up.
